取整

Posted on 2019-08-03 Edited on 2020-02-10 In 算法/数据结构 Comments:

在数据结构的二分查找中常常会遇到的问题，到底是向上取整还是向下取整，向上和向下之间如何进行转化？

除法中的向上取整和向下取整的转化

公式及证明

公式如下：

\[ \left \lfloor (M-1)/N+1 \right \rfloor = \left \lceil M/N \right \rceil \] 证明过程如下：

\(\left \lceil x \right \rceil\)记为\(x\)向上取整，\(\left \lfloor x \right \rfloor\)记为\(x\)向下取整，\(M\)为被除数，\(N\)为除数， \(M,N,k,r\in Z\), \(M\)和\(N\)满足以下关系：

\[ M = kN + r(0<=r<N) \] 若\(r=0\), \(\left \lceil M/N \right \rceil = \left \lceil k \right \rceil = k\), \((M-1)/N=k-1\)$

\(\therefore \left \lfloor (M-1)/N+1 \right \rfloor = k = \left \lceil M/N \right \rceil\)$

若\(r>0\), \(\left \lceil M/N \right \rceil = \left \lceil k+\frac{r}{N} \right \rceil = k+1\),

\(\left \lfloor (M-1)/N+1 \right \rfloor = \left \lfloor (kN+r-1)/N+1 \right \rfloor = \left \lfloor (N+1)+ \frac{r-1}{N} \right \rfloor = N+1\)$

$ (M-1)/N+1 = k+1 = M/N $$

特殊情况

当N为2时，有： \[ \left \lfloor \frac{M+1}{2} \right \rfloor = \left \lceil \frac{M}{2} \right \rceil \]
